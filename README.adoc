# Setup
1. Open the file `~/.gradle/gradle.properties`. (Create it if it does not exist)
2. Add a new entry with the private key you obtained from mapbox.com: `MAPBOX_DOWNLOADS_TOKEN=YOUR_SECRET_MAPBOX_ACCESS_TOKEN`
